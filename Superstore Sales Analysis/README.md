# Superstore Sales Analysis

## Overview

- This Project presents an interactive sales analytics dashboard built using Tableau, based on an Superstore dataset.
- The dashboard provides insights into sales performance, profitability, and customer behavior across different regions.
- It helps users analyze category performance, state-wise sales distribution, customer segments, and sales trends through dynamic and interactive visualizations.

## Objectives

- To analyze overall sales performance across India.
- To identify top-performing product categories and sub-categories.
- To track sales trends over time.
- To evaluate state-wise and region-wise performance.
- To understand customer segment contributions.
- To monitor key KPIs:
    - Total Sales.
    - Profit Margin.
    - Return Rate.
- To enable interactive decision-making using filters.

## Key Insights

- Sales show fluctuations with growth trends over time.
- Technology category contributes the highest revenue.
- Phones and Chairs are leading sub-categories.
- Major states like Maharashtra, Karnataka, and Delhi drive higher sales.
- Consumer segment contributes the largest share.
- Return rate impacts overall profitability.
